# 财神淘宝发布辅助插件

第一版流程：

1. Web 端在“淘宝发布辅助”页面选择整套已通过任务和固定类目。
2. 点击“发布到淘宝草稿”后，任务进入等待插件接收状态。
3. Chrome 插件读取本地 API 地址和连接令牌，自动领取任务。
4. 插件打开淘宝发布页，按任务包填写标题、类目模板默认字段、上传图片并点击保存草稿。
5. 每一步状态会回写 Web 端，失败时保留失败原因。

本地加载方式：

1. Chrome 打开 `chrome://extensions/`。
2. 开启开发者模式。
3. 选择“加载已解压的扩展程序”，目录选择 `D:\自制软件\永沙\apps\taobao-extension`。
4. 打开插件弹窗，填入 Web 端显示的插件连接令牌。

默认 API 地址是 `http://127.0.0.1:8790`。
